Highlights

  • CMS Hospitalization Risk Pipeline: Processes CMS beneficiary and inpatient claims data with a Snakemake workflow and presents hospitalization-risk trends in a Dash application.

  • Healthcare SQL Analytics: Uses Python, DuckDB, and SQL to clean and analyze almost 50,000 healthcare admissions, including readmission and cost metrics.

  • Airflow COVID Mini Pipeline: Demonstrates a reproducible ETL workflow from CDC COVID data to Parquet, DuckDB analytics, and generated reports.

  • Citi Bike Analytics Platform: Combines Citi Bike trips and NOAA weather using Python, DuckDB, dbt, and a Streamlit dashboard.

  • LinkShelf: A Flask bookmark application with PostgreSQL deployment support and optional AI-generated link summaries.
